# 复利投资计算器

一个功能强大、交互友好的复利投资计算器，支持一次性投资和定期定额投资两种方式的计算与可视化对比。

## 功能特点

### 核心计算模块

1. **一次性投资复利计算**
   - 支持设置初始本金、年化利率、投资年限
   - 支持多种复利频率选择（年复利、月复利、周复利、日复利）
   - 计算并展示最终账户金额和总收益率

2. **定期定额投资复利计算**
   - 支持设置每月定投金额、年化利率、投资年限
   - 支持多种复利频率选择（年复利、月复利、周复利、日复利）
   - 计算并展示最终账户金额、总投入本金、总收益和总收益率

3. **投资方式对比分析**
   - 当两种投资方式都计算完成后，自动生成对比图表
   - 直观展示相同条件下两种投资方式的收益差异

### 数据可视化

- 使用Chart.js实现精美的投资增长曲线图
- 支持本金与收益分离显示
- 图表支持动画效果和交互功能
- 可导出图表为图片保存

### 用户体验优化

- 响应式设计，适配各种设备屏幕
- 平滑的交互动画和过渡效果
- 输入验证和错误提示
- 加载状态指示
- 数据变化动画效果

## 使用说明

### 一次性投资计算
1. 在"一次性投资"选项卡中，输入初始本金、年化利率和投资年限
2. 选择复利频率
3. 点击"开始计算"按钮
4. 查看计算结果和投资增长图表

### 定期定额投资计算
1. 在"定期定额投资"选项卡中，输入每月定投金额、年化利率和投资年限
2. 选择复利频率
3. 点击"开始计算"按钮
4. 查看计算结果和投资增长图表

### 投资方式对比
1. 计算完两种投资方式后，系统将自动生成对比图表
2. 对比图表将显示在两种计算结果之后
3. 可以查看不同时间点两种投资方式的收益差异

## 计算公式

### 一次性投资复利公式
```
FV = P × (1 + r/n)^(nt)
```
其中：
- FV = 未来价值（最终金额）
- P = 初始本金
- r = 年化利率（小数形式）
- n = 每年复利次数
- t = 投资年限

### 定期定额投资复利公式
定期定额投资采用模拟计算方法，逐月添加投资金额，并按照设定的复利频率进行复利计算。
